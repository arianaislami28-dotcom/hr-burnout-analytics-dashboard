# 📊 HR Employee Burnout Analytics Dashboard

A full-stack algorithmic analytical tool engineered to track potential employee retention risks by evaluating simulated workplace telemetry.

## 🛠️ The Tech Stack
* **Frontend UI Layout:** React (Vite Engine)
* **Backend Processing API:** Node.js (Express Framework)
* **Cloud Database:** PostgreSQL cluster hosted via Supabase
* **Data Synthesis Engine:** Python script utilizing the Faker data generation utility

## ⚙️ Core Algorithmic Score Parameters
The backend server analyzes operational databases using an explicitly weighted evaluation index framework:
* **Late Night Platform Interactions (40% Weight):** Volatility assessment tracking system usage patterns outside traditional hours.
* **Workplace Velocity Stagnation (30% Weight):** Jira cycle timeline progression checking performance delays.
* **PTO Gaps (30% Weight):** Historical evaluation tracking the duration since the employee last activated vacation time-off.
